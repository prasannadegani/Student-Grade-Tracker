# STUDENT-GRADE-TRACKER

Student Grade Tracker built using **HTML, CSS, and JavaScript** that allows users to enter student marks, calculate total and average, assign grades, determine pass/fail status, and manage student records efficiently.

## Overview

This application helps manage and calculate student academic results and automatically performs:

* Student Marks Entry
* Total Marks Calculation
* Average Calculation
* Grade Calculation
* Pass/Fail Status
* Student Record Management

## Technologies Used

* HTML
* CSS
* JavaScript
* DOM Manipulation

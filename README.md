# STUDENT-GRADE-TRACKER

Student Grade Tracker built using **HTML, CSS, and JavaScript** that allows users to enter student marks, calculate total and average, assign grades, determine pass/fail status, and manage student records efficiently.

## Overview

This application helps manage and calculate student academic results and automatically performs:

* Student Marks Entry
* Total Marks Calculation
* Average Calculation
* Grade Calculation
* Pass/Fail Status
* Student Record Management

## Technologies Used

* HTML
* CSS
* JavaScript
* DOM Manipulation

## Installation

No special installation is required.

Open the project folder in **Visual Studio Code** and run the `index.html` file using **Live Server**.

## Grade System

```text
90% and above → A+
80% - 89%     → A
70% - 79%     → B
60% - 69%     → C
50% - 59%     → D
Below 50%     → F
```

## Assumptions

* Marks are entered between 0 and 100.
* All subject marks are entered before calculation.
* Student information entered is valid.
* Grade is calculated based on average percentage.

## Limitations

* No database connectivity.
* Student records are not permanently stored.
* No user authentication.
* No online student portal.
* Data may be lost when the page is refreshed.

# STUDENT-GRADE-TRACKER

Student Grade Tracker is a web-based application built using **HTML, CSS, and JavaScript** that allows users to manage student records, calculate total marks, average, grade, and pass/fail status efficiently.

## Overview

This application helps manage student academic records and automatically performs:

* Add Student Details
* Calculate Total Marks
* Calculate Average Percentage
* Assign Grade
* Display Pass/Fail Status
* Store Data using Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage
